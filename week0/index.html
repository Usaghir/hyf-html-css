Panguin Adventure

As this game is very simple and I believe anyone can understand easily. This game is about the catching the fishes which are falling from the sky and avoiding the ice cloud, but there are basically six main points need to consider while playing.

How to play this game

Start the game by pressing the green flag on the top right corner above the game screen.

  1. Move the panguin left using the left key.
  2. Move the panguin right using the right key.
  3. Catch as many as falling fishes you can to gain maximum points.
  4. Avoid hitting the falling ice cloud.
  5. Note the number of fishes caught on the top left side of the screen.
  6. Game will finish in case the ice cloud will fall on panguin's head.

Features

This game is very easy to play and have simple moving left and right feature to collect the falling fishes from the sky with a constant speed. Also need to avoid hitting the ice cloud coming down from the sky at random speed. This game has also the ability to count the number of fishes as well as.
